### 轻博客项目

>  一直想做一个博客，但是后台不会写，静态博客？嗯，文笔不太好，目前就只想静静的写东西，当做笔记了自己看看，后期自我感觉有提升的话，就去其他掘金之类的发表一下。

​        本项目后台采用node.js + express 框架，地址为www.lxw-peter.cn(备案成功)。

网站目前确实比较简陋，后期还会装修一番（肯定会的，自己都看不下去了，嘿嘿）

目前实现的功能：

1. 登录/注册（有加密）

2. 分页功能

3. 个人信息（头像上传）

4. 发表文章
5. 编辑文章

列完之后发现竟然如此简单，不行，一定要添加几个功能才行：

 	1. 评论区
 	2. 顶/踩
 	3. 按标签分类
 	4. 按日期分类
 	5. 重新装修页面
 	6. 搜索查询功能
